Batch Processor API documentation

Security Scheme: basic

Type: Basic Authentication
Supports Basic Authentication (RFC2617) for authenticating all API requests

Individual Batch job details

/batch/{batchId} get

get

Returns all the information for the specified batch id

get /batch/{batchId}

Returns all the information for the specified batch id

Secured by basic
  • Request
  • Response

URI Parameters

  • batchId: required (string)

Security Headers

  • Authorization: (string)

    Used to send a username and password encoded token

    Example:

    Basic QWxhZGRpbjpvcGVuIHNlc2FtZQ==

HTTP status code 200

Body

Type: application/json

Example:

{
      "id": "42",
      "status": "COMPLETE",
      "createTime": "2015-02-16T13:23:00Z",
      "updateTime": "2014-05-23T10:36:14Z",
      "endTime": "2014-05-23T10:36:14Z",
      "successCount": 10,
      "errorCount": 1,
      "totalCount": 11,
      "categoryCounts": {
        "created": 3,
        "updated": 4,
        "deleted": 3
      }
    }

Security Responses

HTTP status code 401

Bad or expired token.

HTTP status code 403

Unauthorized request.

/batch/{batchId}/items get

get

Get a list of batch items.

get /batch/{batchId}/items

Get a list of batch items.

Secured by basic
  • Request
  • Response

URI Parameters

  • batchId: required (string)

Security Headers

  • Authorization: (string)

    Used to send a username and password encoded token

    Example:

    Basic QWxhZGRpbjpvcGVuIHNlc2FtZQ==

Query Parameters

  • size: (number)

    The total number of pages to return

  • page: (number)

    The offset of pages to return

  • status: (one of SUCCESS,ERROR)

    Filter items by status.

HTTP status code 200

Body

Type: application/json

Example:

{
          "page": 1,
          "size": 20,
          "total": 2,
          "data": [
            {
              "batchId": "134",
              "itemNumber": 1,
              "status": "SUCCESS",
              "createTime": "2014-05-23T10:35:00Z",
              "updateTime": "2014-05-23T10:36:14Z",
              "category": "new-profile",
              "message": null
            },
            {
              "batchId": "134",
              "itemNumber": 2,
              "status": "ERROR",
              "createTime": "2014-05-23T10:35:00Z",
              "updateTime": "2014-05-23T10:36:14Z",
              "category": null,
              "message": "Missing required email address"
            }
          ]
        }
